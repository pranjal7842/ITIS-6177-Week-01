# ITIS-6177-Week-01

A Simple Node JS App


### Requirements
 - Node
 - Git


### Install
    $ git clone https://github.com/pranjal7842/ITIS-6177-Week-01
    $ cd ITIS-6177-Week-01
    $ npm install


### Starting the Server
    $ node app.js


### Running the project
Hit the URL in browser: http://localhost:3000
